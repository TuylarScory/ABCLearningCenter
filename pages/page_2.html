Course details

Forms are a major component of modern websites, used to collect user data, conduct polls, and more. While you can use a form builder, they're relatively simple to build from scratch with basic web technologies: HTML and CSS. In this course, Clarissa Peterson demonstrates best practices for creating and styling web forms that are easy to complete and acessible to all users and deviecs. Learn proper use of form fields, including the new HTML5 input types, and discover how to add interactivity such as form validation, style forms with CSS, create responsive layouts, add interactivity such as form validation, and design forms for usability and accessibility.

Learning objectives

  • How users interact with forms

  • Designing forms that are easy to use

  • Following data privacy laws when collecting information

  • Choosing with HTML input types to use

  • Adding buttons

  • Styling form fields with CSS

  • Using positioning and flexbox to create responsive form layouts
